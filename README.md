Landing page premium dengan nextjs raect

#### &nbsp Setup:
1. Masuk kedalam direktori
 
2.install depedensi dan menjalankan server
- ```npm install```
- ```npm run dev```
or
- ```yarn```
- ```yarn dev```

3. buka browser: ```http://localhost:3000/```
